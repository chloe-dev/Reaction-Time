Frequently Asked Questions

How accurate is the assessment?


Users providing input via touchscreen will have the lowest accuracy, as the process of registering a user's input introduces latency. Users utilizing high refresh rate monitors may notice an improved reaction time compared to users with lower refresh rate monitors, as the stimulus may be displayed sooner. Why did I fail the assessment?


Strikes are given to users attempting to anticipate the stimulus. Following three strikes, the assessment will prompt the user to restart. This is done to encourage users to react naturally to the stimulus as opposed to anticipating it.
